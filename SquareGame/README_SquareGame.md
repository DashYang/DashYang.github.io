Square Game
============

简介
- 一个基于浏览器的轻量指尖/鼠标益智小游戏，目标是在格子矩阵中找到任意四个角都为同一类型的矩形（2x2 或更大），匹配后获得分数并延长游戏时间。
- 静态文件构成：HTML / JS / CSS，适合直接部署到 GitHub Pages。

玩法要点
- 点击/触摸选中第一个方格，再点击第二个方格作为对角（不能同行或同列）。
- 若四个角的类型相同，则清除矩形范围内方块，得分为行数 * 列数，并奖励额外时间。
- 游戏以时间耗尽结束，结束时会保存分数到本地排行榜（localStorage），可查看排行榜。

技术细节
- 入口：square.html
- 主要脚本：
  - square-config.js：可调的游戏参数（行列、方块大小、初始时间、提示次数等）
  - square-tools.js：地图生成、匹配判定、方块创建/清除、重置/暂停/恢复、提示与排行榜逻辑
  - square-ai.js：帧级动画 / 特效（爆炸动画）
  - square-main.js：主循环、计时与 UI 状态管理
  - gmp-engine.1.7.4.js：轻量游戏对象引擎（G.O API）

扩展功能
（已实现）
1. 改进的爆炸/特效动画（更平滑的缩放与透明度变化）
2. 本地排行榜（localStorage）——游戏结束时自动保存分数并显示前 N 条记录
3. 可视化“提示”功能：高亮一个存在的匹配矩形，并实现有限次数的提示道具

运行与调试
- 直接在浏览器中打开 square.html 即可。建议用本地静态服务器避免某些浏览器对本地文件的限制：
  - Python3: `python3 -m http.server 8000`
